## 概要
ランダムに提示された色がどのくらい好きか、スライドバーで回答していくだけで16Personalities風の結果が出る診断サイト

## 診断の流れ
### 1. ランダムにRGB値が生成され、その色が表示される  
内部でRGB値をランダムに生成します。その色を画面に表示します。
### 2. 好き度をスライドバーで回答する  
どのくらい好き（嫌い）かをスライドバーで表します。
### 3. 10回ごとに回答終了するか続けるかを選ぶ  
最大100回まで回答できます。多く回答するほど正確に診断できます。
### 4. 結果が出る  
16Personalitiesと同様に、ESFP、INTJなどの16タイプに基づく結果と要素ごとの割合が表示されます。

## 診断の仕組み
各要素についてどのように診断しているか説明します。
### 1. 興味関心の方向(E:外向的⇔I:内向的)
日々の生活の中でエネルギーを求める方向を表す「興味関心の方向」は、色の波長をもとに決定しています。波長が長い（赤っぽい）色が好みであるほど外向的であると判断しています。波長が長くなるほど遠くまで届きやすいという物理的な性質があり、色の好みにも影響を与えていると考えました。
### 2. ものの見方(S:感覚型⇔N:直感型)
ものごとに対しどのように結論を出すかを表す「ものの見方」は、彩度で決定しています。彩度が低く現実味のある色が好みであるほど感覚型であると判断しています。
### 3. 判断の仕方(T:思考型⇔F:感情型)
どんなものに価値があると判断するかを表す「判断の仕方」は、色の温度感で決定しています。温度感が低く落ち着いた色が好みであるほど思考型であると判断しています。
### 4. 外界への接し方(J:判断型⇔P:知覚型)
自分以外のものに対しどのような態度をとるかを表す「外界への接し方」は、二成分優勢度で決定しています。二成分優勢度は、R、G、Bのうち、2つの値が残りの1値より突出して大きくなるほど大きくなる値です。二成分優勢度が低くRGB値を予想しやすい色が好みであるほど判断型であると判断しています。
